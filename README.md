## Run Project Locally
Build and Run
```bash
# Build
npm run build
# or
yarn build

# Run
npm run start
# or 
yarn start
```
## Getting Started on Development

First, install npm packages & run the development server:

```bash
# Install packages
npm install
# or 
yarn

# Run Dev Server
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
